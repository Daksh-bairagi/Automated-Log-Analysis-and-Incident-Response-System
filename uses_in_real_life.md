## 2. Cloud Infrastructure Monitoring (DevOps & SRE)
Modern cloud systems generate massive logs from servers, containers, and microservices. AI helps manage and analyze these logs efficiently.

**Uses:**
- Detect server crashes and downtime
- Monitor CPU, memory, and disk usage
- Identify microservice failures
- Auto-restart failed services
- Scale resources automatically during traffic spikes

**Example:**
If error logs and CPU usage spike together, the system restarts the affected service automatically.

## 3. Banking and Financial Systems
Banks and financial institutions rely heavily on logs for security, auditing, and fraud detection.

**Uses:**
- Detect fraudulent transactions
- Monitor suspicious login behavior
- Ensure compliance and auditing
- Detect system failures in payment gateways
- Protect customer data

**Example:**
If a debit card is used simultaneously in two distant locations, the system flags it as fraud and blocks the transaction.

- Monitor hospital IT systems

- Generate audit trails for compliance

If a medical monitoring device logs repeated errors, the system alerts hospital staff immediately.
